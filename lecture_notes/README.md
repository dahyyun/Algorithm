# 📚 Lecture Notes (강의 기반 알고리즘 정리)

이 폴더는 알고리즘 수업, 강의, 학습자료를 기반으로 정리한 노트들입니다.  
개념 이론, 예제 코드, 설명 자료 등을 주제별로 정리하고 있습니다.

---

## ✍️ 주요 정리 방식

- 주차별로 학습한 주제를 분류
- 개념 설명 + 코드 예제 + 주석 포함
- 간단한 퀴즈나 실습도 포함

## 🧠 예제 포맷 예시

- DFS_BasicExample.java : DFS 기본 구현
- BFS_미로탈출.java : BFS 문제 예제 (최단 경로)
- Recursion_개념정리.md : 재귀 함수 흐름 설명
